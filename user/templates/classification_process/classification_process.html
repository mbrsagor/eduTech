Classification Process

Our web-based application leverages sophisticated machine learning and deep learning technologies to seamlessly classify exam questions into Bloom's Taxonomy categories. By analyzing the structure and content of each question, EduBloom ensures that assessments are not just standardized but are also aligned with educational goals and learning outcomes. This automated classification not only enhances the accuracy of determining cognitive levels but also significantly reduces the administrative burden on educators, as it ensures that their assessments are robust, comprehensive, and truly reflective of students’ learning needs.

How EduBloom works?

Step 1 Question Input

Input: Users upload the question in a supported format such as plain text.

Step 2 Question Validation

The system initially verifies if the question aligns with Bloom's Taxonomy grading criteria. If the question does not include valid Bloom's Taxonomy keywords, the system will return the message:

"The question above does not contain any valid Bloom's Taxonomy keywords."
